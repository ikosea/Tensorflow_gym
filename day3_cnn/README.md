# Day 3 – Convolutional Neural Network (CNN)

## 📌 Overview
This project demonstrates how to build a **Convolutional Neural Network (CNN)** using TensorFlow
for **image classification**.

The Cats vs Dogs dataset is used as an example to show how CNNs learn visual patterns
such as edges, shapes, and textures.

> Note: CNNs are not limited to cats and dogs. This dataset is used purely for learning purposes.

---

## 🧠 Concepts Learned
- What a Convolutional Neural Network (CNN) is
- Convolution and Max Pooling layers
- Image preprocessing and augmentation
- Binary image classification
- Training and evaluating a deep learning model

---

## 🛠️ Technologies Used
- Python
- TensorFlow / Keras
- Matplotlib

---

## 🧪 Model Architecture
- Conv2D + MaxPooling (3 blocks)
- Flatten
- Dense (ReLU)
- Output layer (Sigmoid)
