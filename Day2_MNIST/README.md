---

## 🧠 What is MNIST?
MNIST is a dataset of handwritten digits commonly used for learning image classification.

- 60,000 training images  
- 10,000 testing images  
- Image size: **28 × 28 pixels**
- Grayscale images
- Labels: digits **0 to 9**
